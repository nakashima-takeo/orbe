Cmd+, で開く設定パレット（既に開いていれば再フォーカスのみ）。描画は WorkspacePalette／AgentPalette と同じ共有基盤（`PaletteCard`／`PaletteModel`・`AppShell` の `.overlay`）を再利用する。Esc / scrim タップで閉じ、閉じるとアクティブペインへフォーカスが返る。root 行は「スコープ行」と「設定行」の 2 種としてモデル化され、`←→`/`Enter`/`delete` の意味は行 index でなくこの種別と設定レジストリ（root 型は stepper / toggle / drillIn の 3 種）から駆動する——行の並びが変わってもキー操作の意味が壊れないため。

**現在値の解決**: 「現在値」＝**そのスコープの実効値**（グローバル＝global 層、workspace＝global に上書き層を重ねた値）。値の担体はスコープ非依存の均一な設定層で、全設定が一律に上書きできる。デフォルトエージェントだけは表示に起動側の解決規則（設定値が検出済みならそれ、無ければ検出先頭へフォールバック）を通す（実起動と同じキーで見せるため）。root 行の現在値表示・サブパレットの `●`・サブパレットの初期選択ハイライトはすべてこの 1 つの解決だけを読み、**ハイライトはその行で `Enter` を押したときに実際に確定する値（着地点）と常に一致する**。値の表示語彙（`Npt` / `N%` / オン・オフ / テーマ label / フォント名）も 1 箇所のフォーマッタに閉じる。

## root（絞り込み入力欄あり）

先頭にスコープ行、続いて設定行を現在値つきで表示する。↑↓ で行選択（初期選択は先頭設定行＝フォントサイズ）。⌘↑↓ は共有 `PaletteCard` の挙動で有効な先頭/末尾行へジャンプする（無効行＝情報行は飛ばす）。打鍵すると設定行の**ラベルだけ**を増分絞り込みする（現在値・「（継承）」は対象にしない——値が変わると一致が揺れるため）。一致 0 件は情報行「一致する設定がありません」を 1 つ出す（選択・実行対象にしない）。

- **スコープ行（`スコープ <グローバル/この workspace>`）**: `←`/`→`/`Enter` で反転する。パレットは常にグローバルスコープで開く。グローバル＝settings.json（全 workspace 既定）、この workspace＝アクティブ workspace の上書きを編集する（[workspace](workspace.md)・[config](config.md)）。
- **設定行** — `フォントサイズ <N>pt` / `背景の不透明度 <N>%` / `背景のブラー <オン/オフ>` / `カーソルの点滅 <オン/オフ>` / `テーマ <Auto/Dark/Light>`（未設定は Auto 表示） / `デフォルトエージェント <解決済み command>`（検出ゼロで解決不能のときだけ「（未設定）」） / `フォント <名前 or 既定>`（未設定でも既定の実フォント名が読める） / `タブタイトルのフォント <名前 or 既定>`（drillIn） / `絵文字フォント <Noto（同梱）/Apple（システム）>`（未設定は Noto・drillIn） / `エージェントアイコン <既定 or N 状態カスタム>`（状態ごとに個別上書き・drillIn） / `開発中の機能を有効化 <オン/オフ>`。
  - **workspace スコープでの表示**: 全行が上書き可。上書きがあればその値、無ければ global 値を「（継承）」付き・淡色で示す。`delete` で上書きを解除し global 継承へ戻す（絞り込み欄フォーカス中はクエリが**空**のときだけ `delete` が継承解除・**非空**なら文字削除＝backspace。空欄 backspace は元々 no-op なので後退なく相乗りできる）。
  - **グローバルスコープでの上書き注記**: アクティブ workspace が上書き中の行に「（この WS では 16pt）」を付す（画面に効いている値との差が読めるため）。`●`／ハイライト／`Enter` の着地点は global 値のままで、淡色にもしない。注記も「（継承）」も補足スタイル（muted）へ回し、選択・hover でラベルが強調されても主値より弱く読める。行が溢れるときは主値でなく補足側から省略する。
- **フォントサイズ / 背景の不透明度**: 行選択中に `←` 減 / `→` 増（それぞれ 6–72pt、20–100%・端でクランプ）。`settings.json` へ保存し gui.conf 経由でライブ反映（[config](config.md)）。背景の不透明度は surface アルファに加えて host NSWindow の透過（`isOpaque`/`backgroundColor`）も再適用して初めて実際に透ける。端末面だけでなく chrome 各面（StatusRow・パレットのカード・EditorPane）も同じ濃度で透過する（均一ガラス・[layout](layout.md)）。
- **背景のブラー / カーソルの点滅 / 開発中の機能を有効化**: toggle 行（chevron 無し）。行選択中に `←`/`→`/`Enter` のいずれでも反転する（方向でなく flip）。
  - 背景ブラーは gui.conf 再生成＋reloadConfig だけでは効かず、**host が libghostty の C API `ghostty_set_window_background_blur` を自分の NSWindow で能動的に呼んで初めて反映される**（背景の不透明度 <100% かつ非フルスクリーン時のみ・[config](config.md)）。窓背後（デスクトップ）のすりガラスに加え、パレット等のガラス面はブラー ON で VisualEffectView を残し（カード背後の可読性）、OFF で外して素通し半透明にする。
  - 開発中の機能は gui.conf を経由しない UI 表示 gate。効果は右バー（EditorPane）全体の表示可否（→ [editor-pane](editor-pane.md)）で、切替はその場でライブ反映される。未設定時の既定はビルドチャネルで決まり、開発ビルドでオン・公開リリースでオフ。
- **言語（末尾固定・特別行）**: root 末尾に UI 言語のドリルイン行（ラベル `言語 / Language`＋現在言語の endonym。ラベル・値とも現在言語に追従）。設定レジストリを経由せずスコープ非依存（グローバル/workspace で不変）。`Enter`/`→` で ja / en 固定 2 行のサブパレット（絞り込み欄なし）へ潜る。確定は他項目と別経路で提示元へ渡し、「現在言語ストア更新 → AppKit メインメニュー再構築 → 永続化」を即時に束ねる（[localization](localization.md)・[persistence](persistence.md)）。
- **アップデート（言語の次・末尾固定・特別行）**: `アップデート v<現在バージョン>` のドリルイン行。設定レジストリを経由せずスコープ非依存。`Enter`/`→` でアップデートサブパレットへ潜る（→ [update](update.md)）。
- **テーマ / デフォルトエージェント / フォント / タブタイトルのフォント / 絵文字フォント / エージェントアイコン**: `Enter` または `→` でサブパレットへ潜る。

## サブパレット

**共通**: 全行が `● `／2 スペースのプレフィクス規約で列を揃え、`●` は現在値の行にちょうど 1 つだけ付く。潜った直後の選択ハイライトはその `●` の行に乗る（潜る側は選択位置を渡さず、行の再構築で確定した現在値の行を初期選択にする）。絞り込み中は着地点＝先頭一致行にハイライトが移り、`●` は現在値の行に残る。`←`/`Esc` で root へ戻り、潜る前の root 行へ選択が復元される。共有 `PaletteCard` は絞り込みを**入力欄が編集されたときだけ**モデルへ通知するため、モード遷移でモデル自身がクエリを空へ戻しても打鍵として跳ね返らず、置いた初期選択が潰れない。

**フォント（絞り込み入力欄あり）**: macOS の利用可能フォントのうち**等幅 family だけ**を列挙する（CoreText の descriptor 走査で monospace trait が立つ family 名を重複排除・昇順。family 名＝表示名＝選択値＝保存値。列挙不能/0 件は情報行 1 つの空状態）。入力が空のときだけリスト先頭に**解除行**（未設定へ着地する固定行）を出す。解除行はラベルで着地値を名乗る——グローバルなら「既定（<既定の実フォント名>）」、workspace なら「グローバルを継承（<global 値 or 既定>）」。フォント未設定のときの現在値はこの解除行で、`●`＋ハイライトはそこに乗る（カタログ側の同名 family 行には付けない）。`Enter` で確定し gui.conf 経由でライブ反映（[config](config.md)）。gui.conf へは選択フォントをプライマリに据えるため font-family を一度リセットしてから選択＋フォールバックの順で吐く。解除行なら未設定へ戻し、既定チェーンに復帰する。既定の実フォント名は `app/orbe-defaults.conf` の値と同一の SSOT を読む。

**テーマ（入力欄なし・固定 3 択）**: **Auto / Dark / Light**（Auto は OS の外観設定に追従）。確定すると `NSApp.appearance` が同期され chrome とターミナルが揃ってライブ切替（[config](config.md)）。テーマはアプリ全体の外観スイッチで、ターミナル配色自体（Orbe の端末テーマ 2 枚）は選べない。

**絵文字フォント（入力欄なし・固定 2 択）**: **Noto（同梱） / Apple（システム）**（未設定は Noto）。確定で gui.conf の font-codepoint-map 再生成（端末セル）と chrome のフォント解決更新を同時に行いライブ反映（[config](config.md)・[chrome](chrome.md)）。

**タブタイトルのフォント（絞り込み入力欄あり）**: フォントサブと同じ filter 基盤で、**等幅制限なしの全インストール family** を列挙する。入力空のときだけ先頭に解除行。確定は gui.conf を経由せず、chrome のフォント解決器がタブタイトルの描画と幅計測の基底フォントを解決し直配信する（解決不能名は既定へ退避）。任意文字列は `orb config set tab-title-font-family` 経路が受理する（[control-api](control-api.md)）。

**デフォルトエージェント（入力欄なし）**: 検出済みコマンドのみ列挙（起動パレット Cmd+Shift+A と同じ検出結果・[agent-launch](agent-launch.md)）。`●`＋ハイライトは起動パレットと同一の解決規則が返すデフォルトに乗る。検出ゼロは情報行 1 つの空状態。gui.conf は経由せず、実起動（Cmd+Shift+C）はアクティブ WS の実効値を読むため即追従する。パレットは開いた時点の検出結果を固定するため、起動直後（CLI 検出が返る前）に開くとそのセッションだけ空状態になり得る。

**アップデート（入力欄なし・固定 6 行）**: 最上段に状態カード（確認中 / ダウンロード中（進捗）/ 最新です / 失敗（再試行）/ 適用待ち。アップデート状態モデルを直接読みライブに描き替わる——非同期遷移で行は組み直さない）、現在バージョンと最終確認時刻の情報行（選択不可）、トグル 3 行（自動確認 / 自動DL / 終了時自動適用。既定は全オン・`Enter`/`→` で反転）、末尾に「今すぐ確認」。状態カードの `Enter` は第一アクション（適用待ち＝今すぐ再起動 / 失敗＝再試行）、`→` は適用待ちのとき変更内容シートを開く。トグルは設定レジストリ非経由でスコープ非依存（機構と既定・配信は [update](update.md)）。

**エージェントアイコン（2 段ドリル・入力欄なし）**: root の行から潜ると**状態一覧**（`working`/`waiting`/`done`/`idle`/`dormant` の 5 行・各行頭にその状態の実効グリフをプレビュー・`●` は持たない＝値選択でなくナビ行）。状態行で `Enter`/`→` すると**候補アイコン**サブ（先頭に `Glass（既定）`＝解除行、続いて curated SF Symbols・行頭にプレビューグリフ）へさらに潜る。現在の実効アイコンの行に `●`＋初期ハイライトが 1 つ乗る。確定すると状態→SF Symbol 名のマップへ**マップ全体**を書き（workspace スコープでは実効マップをスナップショットして 1 状態だけ差し替えた全マップ・[workspace](workspace.md)・[persistence](persistence.md)）、状態一覧へ戻って編集した状態行に選択が戻る（連続編集できるため）。`←`/`Esc` は 1 段ずつ戻る。反映は gui.conf を経由せず chrome へ env 直配信（[chrome](chrome.md)）。curated 候補は SSOT が 1 箇所。

## 適用の配線（スコープ別）

全設定の変更は「単一代入（値 or nil＝解除）＋スコープ」の形で提示元へ届く。グローバルスコープは in-memory SSOT の global 層へ代入して即 save、workspace スコープはアクティブ workspace の上書き層へ代入（空層は「上書き無し」へ畳む）して workspaces.json へ保存。いずれも最後に実効設定の再適用（gui.conf 再生成→ライブ反映＋外観・状態アイコン・右バー gate の再評価）を呼ぶ（[config](config.md)・[workspace](workspace.md)）。`delete` の上書き解除も同じ経路（当該項目へ nil 代入）。

フォーカスは共有 `PaletteCard` の焦点モデル（カード器と入力欄が常設され、焦点の宛先が常に存在する）に乗り、モード遷移・戻りのたびに確実に移る。
